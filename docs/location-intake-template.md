## Auto-generated fields (do not fill in)

- **id** — derived from name + city (e.g. `valley-orthopedic-specialists-scottsdale`)
- **slug** — same as id, used for URL: `/locations/valley-orthopedic-specialists-scottsdale`
- **lat / lon** — looked up from ZIP code automatically

---

## Before submitting

- [ ] Clinic name is spelled correctly
- [ ] Address is complete and accurate
- [ ] ZIP code is 5 digits
- [ ] Booking URL has been clicked and confirmed working
- [ ] Booking URL starts with `https://`
- [ ] Specialty is one of the three allowed values
- [ ] Treatments supported uses only allowed values (cervical, shoulder, lumbar, hip, knee, ankle)
- [ ] Description is 1-2 sentences, no marketing fluff
